Quick Start: First 7 Days

  • Day 1: Define buy box + profit floor.
  • Day 2: Call 2 hard money lenders and request pre-approval terms.
  • Day 3: Contact insurance broker for vacant property quote ranges.
  • Day 4: Analyze 5 deals with the one-page deal card.
  • Day 5: Submit first 2 offers (target + stretch + walk-away).
  • Day 6: Build contractor shortlist (GC, HVAC, plumbing, electrical).
  • Day 7: Review pipeline + schedule follow-ups.

Step 1: Set Your Buy Box

  • Choose 2-4 zip codes only.
  • Set property type and size limits.
  • Set minimum net profit (ex: $40,000).
  • Set max rehab complexity you can handle.
  • Set max DOM you’re willing to touch.
Rule: If it’s outside your buy box, pass immediately.

Step 2: Financing Setup (Before You Offer)

Checklist: Hard Money Pre-Approval

  • Call at least 2 hard money lenders.
  • Ask for LTV/LTC, points, interest rate, term length.
  • Ask for draw schedule and inspection timing.
  • Ask extension fees and late penalties.
  • Get a written term sheet or email confirmation.

Checklist: Insurance Setup

  • Call insurance broker for vacant flip policy quote.
  • Confirm liability + builder’s risk coverage options.
  • Ask what is required before closing.
  • Get estimated monthly premium for holding cost math.
  • Save broker contact for instant binding at contract stage.

Step 3: Analyze ARV Like a Pro

  • Pull 3-5 sold comps from last 3-6 months.
  • Use similar sqft/style/condition.
  • Ignore inflated active listings.
  • Choose conservative median ARV.
Pro nugget: ARV optimism is the #1 beginner mistake.

  • Add contingency: +10% light rehab, +15-20% heavy rehab.
  • Get at least 2 contractor opinions on bigger items.

Step 5: Full Deal Math

  • Purchase price
  • Rehab
  • Closing costs (buy + sell)
  • Holding costs (interest, taxes, insurance, utilities, HOA)
  • Selling costs (agent/title/concessions)
  • Contingency reserve

Projected Net = ARV - Purchase - Rehab - Closing - Holding - Selling - Contingency

Rule: If net is under your floor, pass.

Step 6: Submit Offer Ladder

  • Set Target Offer
  • Set Stretch Offer
  • Set Walk-away (hard ceiling)
  • Attach short reason: DOM + condition + comp spread
  • Schedule follow-up in 24 hours
Script: “We can close quickly, but only if the numbers stay workable after repairs and carrying costs. Our strongest number is $____.”

Step 7: Under Contract Checklist

  • Order title and lien review.
  • Schedule inspection(s).
  • Lock financing and submit lender docs.
  • Bind insurance policy before close.
  • Finalize scope and contractor calendar.
  • Reconfirm exit strategy (retail + backup).

Step 8: Rehab Management

  • Written scope only.
  • Milestone payments only.
  • Written change orders only.
  • Weekly budget-vs-actual review.
  • Track timeline slippage and carrying cost burn.

Step 9: Sell and Exit

  • Price from sold comps (not wishful pricing).
  • Pro photos + clean listing copy.
  • Set price-reduction trigger date.
  • If stalled, execute backup exit quickly.

Script Bank (Seller / Agent / Lender)

Seller Script

“I can close quickly and keep this simple. Based on repairs, timeline, and comparable sales, my offer is $____. If that works for you, we can move forward today.”

Agent Script

“We are serious buyers. Our number is based on DOM, condition, and spread after rehab. If your seller wants certainty and speed, we can perform.”

Lender Script

“I’m underwriting a flip in [ZIP]. Please send your current terms: rate, points, max LTC/LTV, draw timing, extension fees, and required docs for pre-approval.”

Mistakes + Fix Table
